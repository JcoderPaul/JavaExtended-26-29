******* Запись файлов и класс FileOutputStream *******

Класс FileOutputStream предназначен для записи байтов в файл. Он является производным от класса
OutputStream, поэтому наследует всю его функциональность.

Через конструктор класса FileOutputStream задается файл, в который производится запись.
Класс поддерживает несколько конструкторов:
- FileOutputStream(String filePath) - файл (местоположение) задается через строковый путь;
- FileOutputStream(File fileObj) - файл задается через объект File;
- FileOutputStream(String filePath, boolean append)
- FileOutputStream(File fileObj, boolean append)
Второй параметр - append задает способ записи:
- true - то данные дозаписываются в конец файла;
- false - файл полностью перезаписывается;
Если файла для записи по указанному пути нет, то он автоматически создается при записи.

******* Чтение файлов и класс FileInputStream *******

Для считывания данных из файла предназначен класс FileInputStream, который является
наследником класса InputStream и поэтому реализует все его методы.

Для создания объекта FileInputStream мы можем использовать ряд конструкторов. Наиболее
используемая версия конструктора в качестве параметра принимает путь к считываемому файлу:

- FileInputStream(String fileName) throws FileNotFoundException

Если файл не может быть открыт, например, по указанному пути такого файла не существует,
то генерируется исключение FileNotFoundException.
